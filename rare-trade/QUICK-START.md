## Telegram Commands

### Quick Reference

| Command | Description |
|---------|-------------|
| `/start` | Create new trading bot |
| `/deposit` | Get deposit address |
| `/balance` | Check RARE balance |
| `/strategy` | Configure trading rules |
| `/launch` | Start trading |
| `/pause` | Pause trading |
| `/status` | View performance |
| `/withdraw` | Withdraw funds |
| `/leaderboard` | Top performers |
| `/copy <id>` | Copy a strategy |
| `/premium` | Go private |

## Built-in Strategies

### 🐢 Conservative
- **Timeframe**: Daily
- **Risk**: Low
- **Best for**: Long-term growth
- **Trades/week**: 2-5

### 🦊 Moderate  
- **Timeframe**: 4-hour
- **Risk**: Medium
- **Best for**: Active traders
- **Trades/week**: 10-20

### 🦅 Aggressive
- **Timeframe**: 5-min
- **Risk**: High
- **Best for**: Day traders
- **Trades/week**: 50-100
